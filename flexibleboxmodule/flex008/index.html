Align-self

Align-self não serve para Pais, não serve para conteineres serve para itens. Align-self configurado sobrepõe o valor padrão.

Ela se aplica a cada um dos itens. Funciona diretamente no closs-axis. Cinco valores do Align-self: auto - herda as características de alinhamento vertical de seu pai seja ele o align content ou o align item flex-start - que é o alinhamento perto do cross start flex-end - que o alinhamento perto do cross end center - vai calcular o meio entre o cross start e o cross end stretch - vai dar uma puxada para que ele ocupe o espaço inteiro do eixo transversal.
