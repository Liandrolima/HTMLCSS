Informações sobre o uso do iframe: 1- Conteúdo muito importante evite usar iframe, pois o google bot pode não conseguir entrar em alguns iframe não conseguindo indexar; 2- Não coloque um site inteiro usando iframe, pois pode existir problemas de idetificação; 3 - Usabilidade e acessibilidade, ou seja, pessoas com algum tipo de necessidade especial como por exemplo pessoas cegas, por intermédio de softwares leitores de telas e esses apresentam problemas para acessar iframe. 3- Usabilidade, ou seja, são aqueles problemas em que o próprio navegador acaba se confundindo ou o usuário acaba se confundindo como exemplo: caso se esteja em uma página de um site que usa iframe e se deseja volta para a página principal alguns navegadores apresentam problemas para voltar ou seja não sabem se volta para página principal ou para o iframe; outro exemplo é no uso de targuet _blank alguns navegadores não entendem que é para abrir uma tela nova mas não dentro do iframe; outro problema se o site não for responsivo pode ser que ele apareça todo quebrado. 4- Segurança, ou seja, os iframe abrem as portas para outros site isso tem que ser feito com cuidado devido a ataques, invasão como exemplo croesite, xss
