Como criar um site

Layout do site

Aqui pode se usar a ferramenta mockflow.com não é totalmente gratuita. Essa ferramenta serve para criar estrutura de um site, ou seja, ele vai desenhar, modelar a estrutura do site.

Cores do site

Para cores se pode utilizar o adobe color.

Fontes para o site

pode se usar o google fonts.(nome da fonte: bebas neue). Outra ferramenta para fotes é o dalfont.com(idroid)
