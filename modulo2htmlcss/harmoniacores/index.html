paleta de cores

paleta de cores conjunto de cores que se harmonizão para uma determinada situação, evento, ambiente. em média quatro cores, mas pode ser três a cinco cores

Cores complementares são ascores que tem mais contraste entre si. para saber basta pegar o circulo cromátio e escolher uma cor e traça uma reta para o lado oposto

cores análogas são cores que não tem muito contraste entre elas mas são perceptiveis, são cores meio irmãs. no circulo cromatico escolha uma cor as cores esta do lado que se escolheu são cores análogas.

se pode criar uma paleta de cores como no exemplo a seguir: cores analogas e complementares basta aplicar os conceitos ou seja, escolhe uma cor, as cores do lado serão analogas e traçando uma retas da cor primeiramente escolhida para o outro lado se acha a cor complementar

cores análogas relacionadas são cores que criam um degradê mas não criam contraste. Se se pegar um cor complementar e uma cor análoga cria-se um contraste muito forte. para se ter uma paleta de cores analogas relacionadas no círculo cromátio escolha uma cor outra cor vizinha pule uma cor e escolha outra cor

cores intercaladas são semelhantes as anteriores com contraste amior. para se ter uma paleta neste modo deve se escolher uma cor se pula uma outra cor escolhe outra cor pula novamente e escolhe outra cor. As cores intercaladas principalmente podem ser são de três tipo: 1 - triádicas são cores escolhidas a cada três pulos. 2 - em quadrado, nessa se escolhe a cor depois de cada dois pulos. se se quer usar cores fortes e bem balanceadas. 3- tetradicas são cores complementares. se escolhe por exemplo duas cores e suas respsctivas cores complementares e faz um retângulo delas.

monocromia, ou seja, se escolhe uma cor e altera a saturação e o brilho. Produz efeito degradê. degradê é basicamente modificar saturação e luminosidade de qualquer cor
