Principais elementos de uma caixa

height = altura width = largura border = borda padding = borda interna margin = borda externa outline = borda da borda, traçado, fora do elemento

Tipos de caixa

box-level = sempre se inicia em uma linha nova, 100% da tela do navegador, quebra linha. Ex.:

uso quando quero quebra linha; h1, h2, h3, h4, h5, h6, main, header, nav, article, aside, footer, form, video. inline-level = = não vai pular para outra linha, vai para linha do lado e o mesmo não quebra linha. Ex.: expação de coódigo; span, a, code, small, strong, em, sup, sub, label, button, input, select.
